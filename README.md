# School-Libary-App
A desktop app for borrowing and returning of books for a school library.

## How to Run
1. Download the Windows installer (.exe) from [Releases](https://github.com/Toy085/School-Libary-App/releases).
2. Double-click the installer and follow prompts.
3. Launch the app from your desktop/start menu.
