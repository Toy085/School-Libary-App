A desktop app for borrowing and returning of books for a school library.

## How to Run
1. Download the Windows installer (.exe) from [Releases](https://github.com/Toy085/School-Libary-App/releases).
2. Double-click the installer and follow prompts.
3. Launch the app from your desktop/start menu.

## How to Build
1. Prerequisites
  - Node.js (Version 18 or higher recommended)
  - npm 
  - Git

2. Clone the Repository
Open your terminal or PowerShell and run:
```
git clone https://github.com/Toy085/School-Libary-App.git
cd School-Libary-App
```
3. Install Dependencies
```
npm install
```
4. Build the app
```
npm run build
```
5. Locate the Installer

Once the process finishes, your production-ready files will be in the dist/ folder:

Portable version: dist/win-unpacked/

Installer (.exe): dist/School Library Manager Setup.exe
